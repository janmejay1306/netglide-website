## Installation Steps

1. **Download** the `.dmg` file for your Mac (Apple Silicon or Intel — check  → About This Mac).
2. **Open** the downloaded `.dmg` file.
3. **Drag** the `NetGlide.app` icon into your **Applications** folder.
4. **Eject** the disk image.
5. **Open** NetGlide from your Applications folder or Launchpad.

### First-Launch Security Note

Because NetGlide is not yet notarized with Apple, macOS may show a warning on first launch.
To bypass it:

- **Right-click** (or Control-click) `NetGlide.app` in Applications.
- Choose **Open**.
- Click **Open** in the dialog that appears.

You only need to do this once.
